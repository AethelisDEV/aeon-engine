# 🌌 Aeon Engine

A modular, independent game engine designed for absolute freedom, high-performance execution, and next-gen system architecture.

> **Status:** `aeon-engine` is currently under heavy active development. The initial architecture is being structured locally. The core repository and full engine source code will be officially published alongside the **v1.0.0 Release**.

---

## 🛠️ Tech Stack & Architecture
* **Language:** Rust 🦀
* **Graphics API:** Core architecture tailored for low-overhead modern rendering.
* **Scene Hierarchy:** Modular transform-based Scene Graph system.
* **License:** GNU GPL v3 ⚖️
